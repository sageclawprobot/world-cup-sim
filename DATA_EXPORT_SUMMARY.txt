✅ WHAT'S NEW: CSV DATA EXPORT TOOLS
=====================================

Convert simulator data into 4 CSV formats for analysis:

1. matchup.csv           (Full denormalized data)
2. teams_summary.csv     (Aggregated team statistics)
3. players_summary.csv   (Individual player metrics)
4. match_results.csv     (Goal-by-goal match details)

🔧 CUSTOMIZATION
=================

Add Custom Fields:

  def export_custom(teams, output_path):
      rows = []
      for team in teams:
          for player in team.players:
              row = {
                  "PlayerName": player.name,
                  "Goals": player.goals,
                  "GoalsPerMatch": player.goals / 10,  # Custom
                  "IsFWD": 1 if player.position == "FWD" else 0
              }
              rows.append(row)


Filter Before Export:

  forwards = [p for t in teams for p in t.players 
              if p.position == "FWD"]
  DataExporter.export_players(forwards, "forwards.csv")
